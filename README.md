# XOXOGame
Marmara Üniversitesi Bilgisayar Programcılığı Görsel Programlama I dersi kapsamında geliştirdiğim basit bir oyun. Windows Forms temelini öğrenmek için oluşturuldu.


## Kullanılan Teknolojiler
- **Visual Studio 2022**
- **Scripting:** C#
- **UI Sistemi:** Windows Forms

## Nasıl Oynanır?
1. İlk X ile başlanır.
2. Her adımda sıra diğerine geçer.
3. Eğer düz bir çizgi şeklinde dikey-yatay-çapraz yan yana aynı sembol gelirse o kişi kazanır.
4. Gelmezse ve bütün kutular biterse berabere kalınır.
